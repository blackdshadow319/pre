"지금 우리 강아지
왜 짖는 걸까?"

반려인이라면 누구나 하는 질문입니다.
반가워서? 경계해서? 심심해서?

사람 귀로는 다 비슷하게 들리지만,
소리의 물리적 특성은 분명히 다릅니다.

가장 먼저 짚고 갈 것

이건 "번역기"가 아닙니다

⚠️ 짖는 소리로 강아지의 감정을 정확히 번역하는 것은 과학적으로 불가능합니다.

검증된 데이터셋도, 정답 라벨도 없습니다.
"강아지가 이렇게 말했다"고 주장하는 앱은 대부분 근거가 없습니다.

그래서 이 프로젝트는 측정할 수 있는 것만 측정합니다.
해석은 확신도와 함께, 참고용으로 제시합니다.

특징 추출 ① 음량

RMS — 소리의 세기

파형 값을 제곱해서 평균 내고, 루트를 씌웁니다. (제곱평균제곱근)

// 파형 데이터를 배열로 받아옴
analyser.getFloatTimeDomainData(timeBuf);

let sum = 0;
for (let i = 0; i < timeBuf.length; i++)
    sum += timeBuf[i] * timeBuf[i];   // 제곱해서 더하기

const rms = Math.sqrt(sum / timeBuf.length);  // 평균 → 루트

제곱하는 이유: 파형은 +/− 를 오가서 그냥 더하면 0에 가까워집니다.

특징 추출 ② 음높이

자기상관 — 소리의 주기 찾기

파형을 조금씩 밀면서 자기 자신과 비교합니다.
가장 잘 겹치는 지점이 곧 한 주기의 길이입니다. 

// off 만큼 밀어서 원본과 얼마나 닮았는지 계산
for (let off = MINP; off <= MAXP; off++) {
    let corr = 0;
    for (let i = 0; i < size - off; i++)
        corr += buf[i] * buf[i + off];   // 겹침 정도
    // corr 이 최대인 off = 한 주기의 길이
}
return sampleRate / best;   // 주기 → 주파수(Hz)
탐색 범위 150 ~ 2500 Hz 사람 목소리·생활소음 제외
짖음 분리

어디부터 어디까지가
"한 번의 짖음"일까?

음량이 기준선을 넘으면 시작, 조용해지면 끝. 그 사이를 하나의 이벤트로 묶습니다.

const SILENCE = 0.02;   // 이 아래는 무음으로 간주
const BARK_ON = 0.06;   // 이 위로 올라가면 짖음 시작
const MIN_DUR = 0.08;   // 0.08초 미만은 노이즈로 버림

끝나는 순간 → 평균 음높이 · 최대 음량 · 지속시간 · 최근 3초 반복 횟수
이 4개 숫자가 분류기로 넘어갑니다.

분류 규칙

4개의 숫자 → 5가지 유형

판정근거
🌙외로움 · 불안
하울링 / 낑낑		지속시간 > 0.9초
🎾놀자! · 관심 요구3초 내 3회 이상 & 각 0.35초 미만
😃반가움 · 흥분음높이 > 750 Hz
⚠️경고 · 위협음높이 < 350 Hz & 음량 큼
🐕일반 경계 짖음그 외 (기본값)

각 결과에 확신도 55~72%를 함께 표시해 과신을 방지합니다.

한계

솔직하게 말하면

  • 정답 데이터로 검증하지 못했습니다. 라벨링된 강아지 짖음 데이터셋이 없습니다.
  • 임계값은 경험적입니다. 견종·크기·환경에 따라 조정이 필요합니다.
  • 사람 목소리도 감지될 수 있습니다. 주파수 범위로만 거르고 있습니다.
  • 규칙 기반입니다. 머신러닝이 아니라 if / else 입니다.
